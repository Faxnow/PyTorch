# PyTorch

Этот репозиторий создан для изучения PyTorch. Здесь я собираю чистый код, тестирую базовые операции с тензорами и разбираюсь с оптимизацией памяти.
Код в этом репозитории написан в процессе изучения PyTorch по видео-урокам канала Aladdin Persson.
